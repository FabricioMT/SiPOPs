Novas Features

1 - Nova aba no painel lateral
- descrição de contexto de uso: A nova aba no painel lateral será guia de uso do sistema SPDATA, com informações sobre como usar o sistema e suas funcionalidades com instruções escritas e imagens.
1.1 - Vamos separar os treinamentos por setores, por exemplo:
- Secretaria Urgencia e Emergencia SUS. (deve ser criado uma rota /secretaria-urgencia-emergencia-sus)
    - Procedimentos Para pacientes Externos
    - Procedimentos Para pacientes Internos
- Secretaria Pronto Atendimento. (a rota /health-plans deve ser uma sub rota de /secretaria-pronto-atendimento)
    - Procedimentos Para pacientes Externos
    - Procedimentos Para pacientes Internos
- Secretaria Protaria Principal. (deve ser criado uma rota /secretaria-protaria-principal)
    - Procedimentos Para pacientes Externos
    - Procedimentos Para pacientes Internos

2 - Adicionar a cada plano de saúde os tipos de protocolos de atendimento.
    - descrição de contexto de uso: A cada plano de saúde será adicionado os tipos de protocolos de atendimento, que serão exibidos na tela de cada plano de saúde, abaixo dos guias em Texto vamos criar 2 modais de click para cada tipo de protocolo com instruções escritas e imagens.
    - Procedimentos Para pacientes Externos
    - Procedimentos Para pacientes Internos

3 - Vamos criar novas ROLES, para que possamos diferenciar os usuários por setores.
    - descrição de contexto de uso: As novas ROLES serão usadas para diferenciar os usuários por setores, por exemplo:
    - Secretaria Urgencia e Emergencia SUS.
    - Secretaria Pronto Atendimento.
    - Secretaria Protaria Principal.
    - Secretaria de Central de Guias.
